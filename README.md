# Password Strength Calculator
## Скрипт, который просит ввести пароль и выдаёт ему оценку от 1 до 10. 1 – очень слабый пароль, 10 – очень крутой

# Запуск
```#!bash

$ python3  password_strength.py [password]

```
# Пример вывода скрипта
6
